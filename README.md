สามารถใช้งานโปรแกรมผ่าน monitor.exe ได้เลย!!!
- เพื่อความสะดวกสบายในเวอร์ชันนี้ ผมได้ทำการ complie ออกมาเป็น .exe file ทำให้คนธรรมดาสามารถใช้งานได้เลย โดยการกดเข้าไปที่ monitor.exe แต่สำหรับผู้พัฒนาที่ต้องการพัฒนา project นี้ต่อไป หากคุณไม่ลง library เสริมทั้ง 2 ตัว คือ pyglet และ pillow จะไม่สามารถ run source code ได้ คุณต้องทำการลง libraly ทั้ง 2 ตัว หรือไม่ก็ลบ code ในส่วนที่มี library ทั้ง 2 ตัวออกซะ

เพิ่มโลโก้!!!
- ความเปลี่ยนแปลงอย่างแรกที่เห็นได้ชัด คือ การเพิ่ม logo โรงเรียนเข้าไปเพื่อให้ดูมีความเป็นทางการมากขึ้น

นำปุ่ม "แก้ไขตารางเรียน (ชั่วคราว)" ออก!!!
- ในตอนนี้ผมได้ตัดสินใจที่จะย้ายปุ่ม "แก้ไขตารางเรียน (ชั่วคราว)" ไปไว้ในปุ่มตั้งค่าแทน ซึ่งดูสมเหตุสมผลมากกว่า และได้ทำการเปลี่ยนชื่อไปเป็น ปุ่ม "แลกคาบ" และตอนนี้สามารถใช้งานได้แล้วแต่ยังอยู่ในช่วงทดลองอยู่ Ps. จริงๆแล้วผมออกแบบไว้ว่าจะทำ หน้าแลกคาบ ออกมาเป็นตารางเรียน แล้วสามารถ drag and drop วิชาที่ต้องการจะแลก มาใส่กับอีกวิชาที่ต้องการจะแลกเพื่อสลับคาบได้เลย แต่เนื่องจากอาจารย์มหาวิทยาลัยผมสั่งงานสายฟ้าแลบมา ผมจึงไม่มีเวลามากพอที่จะทำได้ เลยต้องทำตามในแบบที่คุณเห็นไปก่อน

ปุ่มตั้งค่าสามารถใช้งานได้แล้ว!!!
- แน่นอน! จากที่คุณอ่านไปในหัวข้อเมื่อกี้ ปุ่มตั้งค่าสามารถใช้งานได้แล้ว!! แต่มันก็ไม่มีมีอะไรมากมายหรอก นอกจาก 2 ปุ่ม ก็คือ ปุ่ม "แลกคาบ" และ ปุ่ม "คืนค่าตารางเรียน" ในอนาคตอาจจะมีมากกว่านี้แต่ ณ ปัจจุบันนี้เอาไปแค่นี้ก่อน Ps. ปุ่ม "คืนค่าตารางเรียน" นอกจะใช้คืนค่าตารางเรียนแล้วยังสามารถใช้ในการยกเลิกเวลาคืนค่าอัตโนมัติได้อีกด้วย
